Historical National Monuments - Web app

This is a simple JavaScript App that shows 88 Historical National Monuments owned by the French State on a single page. User is then able to click on a Monument card to see details such as its name, image, short description, monument type, city as well as a link to the monument's details on Wikipedia. A search bar also allows the User to find a specific Monument by name. The programming languages are HTML, CSS and JavaScript. The app loads data from an external app also developed by me (cf. links in my GitHub).

Best360ies - REST API

This App is the Backend side of the Best360ies Project. The Best360ies-api is an API that serves 360 panoramic images and users database. User can add their favourite 360 panoramic images into their account. It is a server-side application built with Node.js and Express framework. The API sends responses with information about different 360 panoramic images from the database MongoDB to the client. The connection between the server and the database is implemented using mongoose. As a user you need to create a profile and log in. To ensure data security, an authentication and authorization process is established using JWT Token. Only encrypted paswords (hashed) are stored in the database providing extra protection.

meet App - Serverless, progressive web application (PWA)

The meet App is a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. On the main page, the User is able to enter a City Name and Country. The App returns the upcoming events in the selected city. The User can then decide to see more details on the displayed events by clicking on the relevant button. Optionally, the User can decide the number of events to be displayed (maximum 32). On the main page, a Chart displays the number of events by city. (maximum 32)
